# sam-helper
山姆自动下单，提供捡漏模式和抢单模式，手机端添加购物车，自动下单，去手机端支付既可。

# 说明
开发中，请及时更新代码。

目前只能下单普通商品，暂无法购买全球购等渠道。

请勿长时间使用，会被风控。

抢菜不易，请勿用于商业牟利。

# 使用说明
手机端：app上添加商品至购物车，下单成功后付款。

电脑端：微信小程序，抓包接口数据。以及运行程序。
